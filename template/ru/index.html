monitor.global


monitor.global - система сбора и обработки статистики посещаемости и доступности различных интернет-сервисов, таких как VK.com, Facebook.com, Steam, Twitch.tv, Twitter и т.д. Система собирает и сохраняет данные об онлайне\оффлайне и о количестве онлайн пользователей сервисов 48 раз в сутки. Система позволяет получить следующие статистические данные о любом сервисе, как то:

  • данные об онлайне\оффлайне сервиса за промежуток времени (день \ неделя \ месяц)
  • данные о количестве онлайн пользователей сервиса за промежуток времени (день \ неделя \ месяц)
  • данные о сравнении количества онлайн пользователей для выбранного сервиса за разные промежутки времени (дни \ недели \ месяцы)
  • данные о сравнении количества онлайн пользователей между любыми двумя сервисами за любые промежутки времени

Для удобства, все сервисы разбиты на категории.

Реализована поддержка многоязыкового интерфейса
При разработке данной системы были использованы возможности следующего ПО, технологий и библиотек:

  • apache
  • php
  • mysql
  • ajax
  • html\css\javascript
  • php библиотека pChart (частично переписанная)
  • php библиотека jChart
  • php библиотека xmpphp (частично переписанная)
  • swf библиотека Open Flash Chart
  • javascript библиотека highslide (частично переписанная)
  • linux cron

Объем кода ~ 12К строк.
